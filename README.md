# discover-webpack
This project is for a presentation at Human Talks Grenoble.

It uses:

* webpack
* react
* babel
* eslint

## Getting started

```bash
$ yarn install

# to generate a heavy json database
$ yarn weight

$ yarn start

# to build with a production environment
$ yarn build

# to serve built files with a http server
$ yarn serve

```
